# Unity 프로젝트 설정 가이드 (2D Wave Shooter)

이 문서는 프로젝트의 핵심 시스템을 유니티 에디터에서 설정하는 방법을 설명합니다.

## 1. 적(Enemy) 시스템 설정
적이 화면 상단에서 자동으로 생성되어 내려오게 하려면 다음 단계를 따르세요.

### A. 적(Enemy) 프리팹 만들기
1. **오브젝트 생성**: Hierarchy 창 우클릭 -> 2D Object -> Sprites -> Square (사각형 생성).
2. **이름 변경**: 생성된 오브젝트 이름을 `Enemy`로 변경.
3. **컴포넌트 추가** (Inspector 창에서):
   - `Enemy.cs` 스크립트 추가.
   - `Box Collider 2D` 추가 후 **Is Trigger** 체크 (충돌 감지용).
4. **프리팹화**: Hierarchy의 `Enemy`를 프로젝트 창의 `Assets/Prefabs` 폴더로 드래그 앤 드롭.
5. **정리**: 프리팹을 만들었다면 Hierarchy에 있는 `Enemy`는 **삭제**합니다. (공장에서 찍어낼 '설계도'가 생겼으므로 원본은 지워도 됩니다.)

### B. 적 생성기(EnemySpawner) 설정
1. **공장 생성**: Hierarchy 창 우클릭 -> Create Empty (빈 오브젝트 생성).
2. **이름 변경**: 이름을 `EnemySpawner`로 변경.
3. **스크립트 추가**: `EnemySpawner.cs` 스크립트를 추가.
4. **설계도 연결**: Inspector 창의 `Enemy Prefab` 칸에 아까 만든 **Enemy 프리팹**을 드래그하여 할당합니다.
   - **주의**: 이 오브젝트는 Hierarchy에 **남겨두어야** 합니다. 게임 실행 중에 적을 계속 찍어내는 '공장' 역할을 하기 때문입니다.

## 2. 용어 정리
- **Hierarchy(하이어라키)**: 현재 씬(화면)에 나와 있는 실제 물체들.
- **Project(프로젝트)**: 게임에 쓰일 모든 파일(설계도, 이미지, 코드 등) 보관함.
- **Prefab(프리팹)**: 미리 만들어둔 오브젝트의 '설계도'. 언제든 복제해서 쓸 수 있음.
- **Is Trigger**: 물리적인 부딪힘(튕겨 나감) 없이 '겹쳐졌음'만 감지할 때 사용.

## 3. 실행 결과 확인
- 게임 실행(Play)을 누르면 Hierarchy 창에 `Enemy(Clone)`들이 자동으로 생겨나는지 확인하세요.
- 만약 안 보인다면 `EnemySpawner`의 `Enemy Prefab` 칸이 비어있는지(None) 확인하세요.

## 4. 충돌 처리(Collision) 설정 및 작동 원리
총알이 적을 맞췄을 때 적이 파괴되는 로직은 `OnTriggerEnter2D`라는 유니티 기능을 사용합니다.

### A. 작동 원리 (코드 관점)
1. **감지**: 두 물체가 겹치는 순간 유니티 엔진이 `OnTriggerEnter2D` 메서드를 자동으로 실행합니다.
2. **필터링**: 충돌한 상대방의 `Tag`가 `"Enemy"`인지 확인합니다.
3. **실행**: 맞다면 적의 `Die()` 메서드를 호출하여 적을 제거하고, 총알 자신도 `Destroy()` 합니다.

### B. 유니티 에디터 필수 설정 (이걸 안 하면 코드가 안 돌아갑니다!)
1. **적(Enemy) 프리팹 태그 설정**:
   - `Enemy` 프리팹을 클릭하고 Inspector 상단의 **Tag**를 **`Enemy`**로 변경합니다. (목록에 없으면 `Add Tag...`로 직접 추가해야 함)
2. **Collider(충돌체) 설정**:
   - `Enemy`와 `Bullet` 프리팹 모두 **`Box Collider 2D`**가 있어야 합니다.
   - 두 오브젝트 모두 **`Is Trigger`** 체크박스가 **체크**되어 있어야 합니다. (체크 안 하면 물리적으로 튕겨나갑니다.)
3. **Rigidbody 2D (매우 중요!)**:
   - 유니티에서 충돌을 감지하려면 충돌하는 두 물체 중 최소 하나에는 **`Rigidbody 2D`** 컴포넌트가 있어야 합니다.
   - **설정 방법**: `Enemy` 프리팹에 `Rigidbody 2D`를 추가하고, **`Gravity Scale`**을 **0**으로 설정하세요. (0으로 안 하면 적이 중력 때문에 아래로 툭 떨어집니다.)

## 5. 자주 발생하는 문제 해결 (FAQ)
- **질문: 총알이 적을 그냥 통과해요!**
  - 답변 1: 적 프리팹의 **Tag**가 정확히 `Enemy`인지 확인하세요. (대소문자 구분)
  - 답변 2: 두 물체 중 하나에 **Rigidbody 2D**가 있는지, 그리고 **Is Trigger**가 체크되어 있는지 확인하세요.
- **질문: 적이 소환되자마자 아래로 떨어져요!**
  - 답변: `Enemy`에 붙은 `Rigidbody 2D`의 **Gravity Scale**을 **0**으로 바꾸었는지 확인하세요.
