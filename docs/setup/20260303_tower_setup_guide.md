# 타워 시스템 설정 마스터 가이드 (2026-03-03)

이 문서는 기존의 이동형 슈팅 게임에서 고정형 타워 디펜스 스타일로 게임 디자인을 변경함에 따라, 자동 사격 타워를 유니티에서 설정하는 방법을 상세히 설명합니다.

## 1. 타워(Tower) 프리팹 만들기 (BasicTower)
1. **오브젝트 생성**: Hierarchy 우클릭 -> **2D Object** -> **Sprites** -> **Square** 클릭.
   - 이름 변경: **`BasicTower`**
2. **스크립트 추가**: 프로젝트 창(`Assets/Scripts/`)에 있는 **`Tower.cs`**를 `BasicTower` 오브젝트에 드래그하여 추가합니다.
3. **색상 변경**: Inspector의 `Sprite Renderer`에서 Color를 원하는 색(예: 파란색)으로 변경하여 플레이어와 구분합니다.

## 2. 발사구(FirePoint) 설정 (매우 중요!)
타워의 자식 오브젝트로 총알이 나갈 '총구' 위치를 만들어야 합니다.
1. **자식 오브젝트 생성**: Hierarchy에서 `BasicTower` 위에서 **우클릭** -> **Create Empty** 클릭.
   - 이름 변경: **`FirePoint`**
2. **위치 잡기** (Inspector의 `Transform` 섹션):
   - **X: 0, Y: 0.6, Z: 0**으로 입력합니다. 
   - 이 수치는 1x1 타워 몸통의 **살짝 위쪽 끝**을 의미합니다.
3. **스크립트 연결**: 부모인 **`BasicTower`**를 클릭하고 Inspector의 `Tower` 스크립트 섹션을 봅니다.
   - **`Fire Point`** 칸에 방금 만든 자식인 **`FirePoint`**를 드래그해서 넣습니다.
   - **`Bullet Prefab`** 칸에 프로젝트 창의 **`Bullet`** 프리팹을 드래그해서 넣습니다.

## 3. 사거리(Range) 조절 및 시각화
타워가 적을 감지하는 범위를 확인하고 수정하는 방법입니다.
- **노란색 원 확인**: `BasicTower`를 클릭한 상태에서 `Scene` 뷰를 보면 노란색 원이 보입니다. (Gizmos가 켜져 있어야 함)
- **사거리 변경**: Inspector의 **`Range`** 숫자를 키우거나 줄이면 노란색 원의 크기가 실시간으로 변합니다. (기본값: 5.0 권장)
- **발사 속도**: **`Fire Rate`**는 초당 발사 횟수입니다. (기본값: 1.0 = 1초에 한 번 발사)

## 4. 작동 원리 (Troubleshooting)
- **총알이 안 나가요!**: 
  - `GameManager`가 씬에 배치되어 있는지 확인하세요.
  - `Tower` 인스펙터에 `Bullet Prefab`과 `Fire Point`가 모두 연결되어 있는지 확인하세요.
  - 적(`Enemy`)이 타워의 노란색 사거리 원 안으로 들어오고 있는지 확인하세요.

## 5. 사격 로직 최적화 (즉시 발사 시스템)
타워의 인지 속도를 높이기 위해 다음과 같은 로직이 적용되어 있습니다.
- **인지 즉시 사격 (Instant First Shot)**:
  - 기존 로직: 적 발견 -> 쿨타임 대기 (1초) -> 발사. (체감상 인지가 느리다고 느껴짐)
  - 개선 로직: 적 발견 -> **즉시 첫 발 발사** -> 쿨타임 대기 (1초) -> 후속 발사.
- **설계 원리**: 타겟이 없을 때 타워의 사격 타이머를 항상 '발사 준비 완료' 상태로 유지합니다. 이를 통해 적이 사거리 안에 들어오는 찰나에 바로 반응할 수 있습니다.

## 6. 타격감 개선 팁 (UX Optimization)
- **발사 속도(`Fire Rate`)**: 2.0 ~ 3.0 정도로 올리면 타워가 훨씬 민첩해 보입니다.
- **총알 속도(`Bullet Speed`)**: `Bullet` 프리팹의 Speed를 25 이상으로 올리면 총알이 적에게 꽂히는 느낌이 강해집니다.
- **사거리(`Range`)**: 타워가 너무 늦게 쏘는 것 같으면 사거리를 7.0 정도로 키워보세요. 화면 상단에서 적이 나타나자마자 인지하고 쏘기 시작합니다.
