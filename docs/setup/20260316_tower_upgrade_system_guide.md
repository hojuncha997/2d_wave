# 타워 업그레이드 시스템 가이드 (2026-03-16)

이 문서는 `2d_wave` 프로젝트에 구현된 **타워 업그레이드 시스템**의 구조와 사용 방법을 설명합니다.

---

## 1. 시스템 개요
플레이어는 이미 설치된 타워를 클릭하여 골드를 소모하고, 해당 타워의 성능(공격력, 사거리, 공격 속도)을 영구적으로 강화할 수 있습니다. 이는 제한된 슬롯 내에서 효율적으로 방어력을 높이는 전략적 요소입니다.

## 2. 기술적 구현 상세

### 2.1 Bullet.cs (탄환 로직)
타워의 강화된 공격력을 실제 적에게 전달하기 위해 데미지 설정 기능을 추가했습니다.
- `SetDamage(int damage)`: 타워가 발사 시점에 자신의 현재 데미지를 총알에 주입합니다.

### 2.2 Tower.cs (타워 로직)
타워의 현재 레벨과 능력치 상태를 관리하고 강화 로직을 처리합니다.
- **주요 필드**:
    - `_level`: 현재 타워의 레벨 (기본 1).
    - `_damage`: 적에게 입히는 데미지.
    - `_upgradeCost`: 다음 레벨업에 필요한 골드 (기본 75G).
    - `_upgradeMultiplier`: 레벨업 시 능력치 상승 폭 (기본 1.2배).
- **Upgrade() 메서드**:
    - 호출 시 레벨이 1 증가합니다.
    - **공격력**: 1.2배 상승 (정수 반올림).
    - **사거리/연사 속도**: 각 1.1배 상승.
    - **업그레이드 비용**: 다음 레벨을 위해 1.5배 상승.

### 2.3 TowerSlot.cs (상호작용 로직)
플레이어의 클릭 입력을 감지하여 '설치' 또는 '업그레이드'를 판단합니다.
- **OnMouseDown()**:
    - 슬롯이 비어있을 경우: 타워 설치 (50G).
    - 타워가 이미 있을 경우: `GameManager`를 통해 골드 잔액을 확인한 뒤 `Tower.Upgrade()` 호출.

## 3. 업그레이드 공식 및 밸런스
현재 설정된 기본 밸런스 수치는 다음과 같습니다.

| 능력치 | 초기값 | 레벨업 배율 | 비고 |
| :--- | :--- | :--- | :--- |
| 레벨 | 1 | +1 | - |
| 공격력 | 1 | x1.2 | `Mathf.RoundToInt` 적용 |
| 사거리 | 5.0 | x1.1 | - |
| 연사 속도 | 1.0 | x1.1 | 초당 발사 횟수 |
| 업그레이드 비용 | 75 | x1.5 | - |

## 4. 사용 방법 및 주의사항
1. **업그레이드 확인**: 게임 실행 중 설치된 타워를 클릭하면 콘솔(Console) 창에 현재 레벨과 강화된 수치가 로그로 표시됩니다.
2. **골드 부족**: 골드가 부족한 상태에서 클릭하면 업그레이드가 진행되지 않으며 `GameManager`에서 경고 로그가 발생합니다.
3. **수치 조정**: `Tower` 프리팹의 인스펙터 창 내 `Upgrade Settings` 섹션에서 배율과 초기 비용을 자유롭게 수정할 수 있습니다.
