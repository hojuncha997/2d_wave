# 플레이어 체력 및 게임 관리자 설정 (2026-03-03)

이 문서는 플레이어의 HP 시스템과 게임 종료/재시작 기능을 유니티 에디터에서 설정하는 방법을 설명합니다.

## 1. 플레이어 체력 설정 (Player HP)
플레이어가 적과 부딪혔을 때 체력이 깎이도록 설정합니다.

### A. 플레이어 오브젝트 설정
1. **오브젝트 선택**: Hierarchy 창에서 `Player` 오브젝트를 선택합니다.
2. **스크립트 확인**: `PlayerController.cs`가 붙어 있는지 확인합니다.
3. **체력 수치 조정**: Inspector 창의 **Health** 섹션에서 `Max Hp` 값을 원하는 만큼(기본 3) 조정합니다.
4. **태그 설정**: 플레이어 본인의 Tag는 굳이 `Player`일 필요는 없지만, **Enemy** 프리팹이 이 플레이어를 인식하려면 적절한 태그와 콜라이더가 필요합니다.

### B. 충돌 설정
- 플레이어에게도 **`Box Collider 2D`**가 있어야 하며, **`Is Trigger`**가 체크되어 있어야 합니다.
- 플레이어가 적과 부딪히면 콘솔 창에 `플레이어 체력: 2/3`와 같은 메시지가 뜨는지 확인하세요.

## 2. 게임 관리자 설정 (GameManager)
전체 게임의 흐름을 제어하는 '뇌' 역할을 하는 오브젝트를 만듭니다.

### A. GameManager 오브젝트 생성
1. **빈 오브젝트 생성**: Hierarchy 우클릭 -> Create Empty.
2. **이름 변경**: 이름을 `GameManager`로 변경합니다.
3. **스크립트 추가**: `GameManager.cs` 스크립트를 드래그해서 넣습니다.
4. **위치**: 보통 `(0, 0, 0)`에 두어 관리하기 편하게 합니다.

### B. 게임 오버 및 재시작 테스트
1. **사망 테스트**: 적에게 일부러 3번 부딪혀 봅니다.
2. **콘솔 확인**: 플레이어가 사라지면서 `"게임 오버! 'R'키를 눌러 재시작하세요."`라는 로그가 뜨는지 확인합니다.
3. **재시작**: 키보드 **'R'**키를 눌렀을 때 게임이 다시 시작(씬 리로드)되는지 확인합니다.

## 3. 요약: 게임 루프 완성
이제 우리 게임은 **[시작 -> 생존/전투 -> 죽음 -> 재시작]**이라는 완벽한 한 주기의 루프를 갖추게 되었습니다!
