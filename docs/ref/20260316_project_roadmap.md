# 프로젝트 로드맵 (2026-03-16 업데이트)

이 문서는 `2d_wave` 프로젝트의 현재 진행 상황과 향후 개발 계획(To-Do)을 정리한 로드맵입니다.

---

## 1. 현재 완료된 핵심 기능 (Done)
- [x] **기지(Base) 시스템**: `BaseManager`를 통한 기지 체력 및 게임 오버 로직.
- [x] **타워(Tower) 시스템**: 자동 타겟팅, 즉시 사격, 탄환 발사 로직.
- [x] **경제 시스템**: `GameManager`를 통한 골드 획득 및 소비 기능.
- [x] **타워 배치**: `TowerSlot`을 통한 골드 소비 및 타워 건설 인터랙션.
- [x] **동적 난이도**: 시간에 따른 적 생성 속도 및 마리수 증가 시스템.

---

## 2. 단기 목표: 게임 플레이 다양화 (To-Do - High)
- [ ] **웨이브 시스템 (Wave System)**: 
    - 단순 시간 경과가 아닌, 명확한 'Wave 1, Wave 2' 단위의 스테이지 구성.
    - 각 웨이브마다 등장하는 적의 수와 종류 정의.
    - 모든 적 처치 시 다음 웨이브 준비 시간(Intermission) 제공.
- [ ] **보스 시스템 (Boss System)**:
    - 특정 웨이브(예: 5, 10웨이브) 마지막에 등장하는 거대 보스.
    - 높은 체력, 고유한 공격 패턴 또는 특수 능력.
- [ ] **타워 종류 확장**:
    - **AOE Tower**: 범위 공격을 하는 폭발형 타워.
    - **Slow Tower**: 적의 이동 속도를 늦추는 유틸리티 타워.
    - **Pierce Tower**: 여러 적을 관통하는 레이저형 타워.
- [ ] **타워 업그레이드**: 설치된 타워를 클릭하여 골드로 레벨업(데미지, 사거리 증가).
- [ ] **적 유닛 다양화**:
    - **Tanker Enemy**: 체력은 높지만 속도가 느린 적.
    - **Speedy Enemy**: 체력은 낮지만 속도가 매우 빠른 적.

---

## 3. 중기 목표: 시스템 고도화 및 최적화 (To-Do - Medium)
- [ ] **오브젝트 풀링 (Object Pooling)**: 총알(`Bullet`)과 적(`Enemy`)의 잦은 생성/파괴로 인한 성능 저하 방지.
- [ ] **UI 개선**:
    - 타워 건설 시 비용 표시 팝업.
    - 기지 체력바(HP Bar) 시각화.
    - 일시정지(Pause) 및 설정 메뉴.
- [ ] **카메라 제어 (Camera Control)**:
    - 마우스 휠 또는 핀치 줌을 통한 화면 확대/축소.
    - 확대 시 화면 드래그를 통한 시점 이동(Panning).

---

## 4. 장기 목표: 폴리싱 및 완성도 (To-Do - Low)
- [ ] **시각 효과 (VFX)**: 적 피격 시 스파크, 타워 건설 먼지 효과, 기지 파괴 폭발.
- [ ] **사운드 효과 (SFX)**: 발사음, 폭발음, 골드 획득음, 배경음악(BGM).
- [ ] **웨이브 시스템**: 단순 시간 경과가 아닌, 명확한 'Wave 1, Wave 2' 단위의 스테이지 구성.
